# NAVERCloudCamp
네이버 클라우드 캠프 관련 자료, 학습 내용을 기록합니다.

공부
```
* 클라우드 기반 웹프로그래밍
  - Cloud 기술 이해
  - NCP IaaS 구성 및 활용
  - JavaScript, React, Java, Spring

* DevOps 프로그래밍
  - SaaS API
  - NAVERCloud Platform
  - NCP DB & Storage
  - Linux
  - Docker

* MSA 기반 프로그래밍
  - kubernetes
  - Spring Cloud
  - Spring 배치, Spring 시큐리티
```

프로젝트
```
* 클라우드 기반 웹 프로젝트
   - 게시판 서비스 구축
   - 외부 API와 공공데이터 활용한 서비스
   - PT 이용권 관리 서비스

* DevOps 프로젝트
   - 코드를 통안 인프라 관리 프로젝트
   - CI/CD 구현 프로젝트
   - 애플리케이션 성능테스트 및 모니터링 프로젝트

* MSA 기반 프로젝트
   - MSA 기반 구글 캘린터 클론 프로젝트
   - MSA 기반 P2P 대출 서비스 개발 프로젝트
   - MSA 기반 대용량 데이터 처리 프로젝트	
```
